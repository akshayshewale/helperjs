array getValuesByNames()

Description This function returns an array consisting of values of names as given in the input array
*Unlike getValuesByIds,which retrieves data regardless of the ids value,The getValuesByNames wont retrieve data from a name if it is empty

Parameters
SrNo. name Description Datatype is Optional
1 arrayOfNames it is the array of names from which you want to retrieve values from array no

Code
Language:JS
let arrayOfNames = ["nameGetValuesByNames","phoneGetValuesByNames","emailGetValuesByNames","divExampleGetValuesByNames"];
document.getElementById("outputGetValuesByNames").innerHTML = getValuesByNames(arrayOfNames);
