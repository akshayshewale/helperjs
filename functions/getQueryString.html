String getQueryString()

Description This is a function that returns a created query string according to the passed parameters
*This Does NOT Request the url

Parameters
SrNo. name Description Datatype is Optional
1 file it is the file you want to request/make query string/url of String no
2 headerOfIds it is the Header name for the corresponding id [?header=value] array no
3 arrayOfIds it is the array of ids that you want to retrieve data from and store into the header corresponding to its index [?header=(valueOfId)] array no
4 headerOfNames it is the Header name for the corresponding name [?header[]=value] array yes
5 arrayOfNames it is the array of names that you want to retrieve data from and store into the header corresponding to its index [?header[]=(valueOfName)] array yes

Language:JS
