boolean validator()

Description This function validates and return true if the passed value is of the given type

Parameters
SrNo. name Description Datatype is Optional
1 type it is the type to be used for validation.some Types are isNotNull,phone,email,name,number,checkbox or radio String no
2 value it is the value to be validated(provide name in case of radio or checkbox else provide value) String,int no

Code
Language:JS
let type = document.getElementById("typeValidator").value;
let name = document.getElementById("nameValidator").value;
let isTrue = validator(type, name);
if (isTrue) {
document.getElementById("outputValidator").innerHTML = name + " is a " + type;
} else {
document.getElementById("outputValidator").innerHTML = name + " is NOT a " + type;
}
