void include()

Description This is an implicitly called function that includes specified files into given ids in the include tags
*this just includes the file to the page containing the include tag,so any references/links given in the file should be with respect to the parent file
*you can also use the base tag to set the base as the parent file for relative linking

Parameters none

Code
Language:HTML
<include file="fileName" to="idName"><include>
<div if="idName"><div>
<!--this includes the file.extension contents to the given id in this case itself-->
Sample
Language:HTML
<include file="file.html" to="storeId" id="storeId"><include>
<!--this includes the file.html contents to the given storeId in this case itself--> <!--for a better sample view source code of this page-->
Events Two Events are fired when running the include() function
*includestart :: this event is fired when the including process has started
*includednew :: this event is fired every time a new file is finished including to know the id of the div where the file is use (event.id)
*includeend :: this event is fired when the including process has ended
(use the end event to initialize any code meant for files in the include tag
{as window.onload runs when the main file is loaded and it does not care for include to finish})
