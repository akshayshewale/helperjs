void include()

Description This is an implicitly called function that includes specified files into given ids in the include tags
*this just includes the file to the page containing the include tag,so any references/links given in the file should be with respect to the parent file
*you can also use the base tag to set the base as the parent file for relative linking

Parameters none

Code
Language:HTML
<include file="fileName" to="idName"><include>
<div if="idName"><div>
<!--this includes the file.extension contents to the given id in this case itself-->
Example
Language:HTML
<include file="file.html" to="storeId" id="storeId"><include>
<!--this includes the file.html contents to the given storeId in this case itself-->
