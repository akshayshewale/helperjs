boolean checkRequiredNames()

Description This function returns an boolean which is true if at least one value of the name is filled
*Unlike checkRequiredIds,which checks each id and return false if even one is empty,The checkRequiredNames check if ATLEAST one name is filled else returns false

Parameters
SrNo. name Description Datatype is Optional
1 arrayOfNames it is the array of names from which you want to retrieve values from array no

Code
Language:JS
let arrayOfNames = ["nameCheckRequiredNames","phoneCheckRequiredNames","emailCheckRequiredNames","divExampleCheckRequiredNames"];
document.getElementById("outputCheckRequiredNames").innerHTML = checkRequiredNames(arrayOfNames);
