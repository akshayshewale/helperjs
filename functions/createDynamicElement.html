void createDynamicElement()

Description this function adds a dynamic element into a div whose id is provided on calling function
it replaces given string(default "##") with numbers,and replicates the first child template div of the given id.

Parameters
SrNo. name Description Datatype is Optional
1 parentId The id in which you want the dynamic element to be created String no
2 replaceStringPattern The string pattern to be replaces from the template div(Default "##") String yes

Code
Language:HTML
<div id="makeDynam">
<div>
<div class="col s12 m6 l6 input-field">
<input type="text" id="nameCreateDynamicElement##" name="nameCreateDynamicElement">
<label for="nameCreateDynamicElement##">Name ##</label>
<!--You can add any replacestring anywhere in the template--> </div>
</div>
</div>
 Language:JS
//needed to be called once if template is not hidden or if you want to keep x number of items pregenerated createDynamicElement("makeDynam");
